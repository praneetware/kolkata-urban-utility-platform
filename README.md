# Kolkata Urban Analytics & Infrastructure Load Testing Platform

An interactive, enterprise-grade data intelligence dashboard built to visualize historical census trends, simulate long-term population growth, and load-test municipal utility capacities for Kolkata through 2050. 

This platform leverages data from the **District Census Handbook (DCHB) Town Amenities dataset** to bridge static historical records with dynamic, real-time predictive modeling.

## 🚀 Key Features Demonstrated in the Video

* **Centennial Growth Curve:** Tracks and displays Kolkata's official decadal population trajectory across an entire century (1901–2011).
* **Multi-Sector Utility Load Testing:** A dynamic forecasting matrix that calculates future stress vectors for the city's infrastructure:
  * **Water Supply Network:** Tracks volumetric daily demands (MLD) mapped against baseline reservoir thresholds.
  * **Electrical Grid Overhead:** Simulates domestic/commercial demand scaling in Megawatts (MW) against power distribution caps.
* **20-Year Property Tax Revenue Predictor:** A financial simulation model tracking future municipal revenue pools generated by compounded household expansion variables.
* **⚡ Smart City Policy Toggle:** A master scenario switch that automates controlled urban migration thresholds, drops hazardous residential overcrowding, simulates closed-drainage upgrades, and applies smart grid efficiency factors (-12% water wastage, -15% power load optimization, and +15% tax compliance efficiency).
* **Automated Executive PDF Export:** Clean client-side reporting architecture powered by `html2pdf.js` to package active simulations into a print-ready, one-page executive dossier.

## 🛠️ Tech Stack
* **Frontend Structure:** HTML5, Tailwind CSS
* **Data Visualization:** Chart.js Engine (Dynamic multi-chart synchronization pipeline)
* **Client-Side Export:** html2pdf.js
* **Data Foundation:** Government of India DCHB Town Amenities Registry
